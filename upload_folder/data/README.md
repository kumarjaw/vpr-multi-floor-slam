# Dataset

This project uses a custom dataset of 1,331 images captured across 
multiple floors of Northeastern University's ISEC building.

## Contents
- **2nd floor**: Images from the 2nd floor corridors and common areas
- **5th floor**: Images from the 5th floor corridors and common areas

## Why not included?
The full image dataset is ~18GB and too large for GitHub. 
The pre-computed descriptors (.npz files) are included so you can 
reproduce the evaluation and training steps without the raw images.

## Metadata
- `isec_2f_5f_metadata.xlsx` — Image labels, floor assignments, 
  and capture metadata
